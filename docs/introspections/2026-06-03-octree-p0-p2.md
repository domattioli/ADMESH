## Key decisions

- **Dict-keyed neighbours** (`{"N", "S", "E", "W"}`) over int-index list: clearer intent, no off-by-one for direction lookups
- **`OctreeGrid = OctreeTree` alias** preserved for `octree_medial.py` backward compat without touching the locked stage
- **Adaptive refinement gate** (`d > 2.0 * node.size` → skip): avoids refining exterior cells; keeps leaf count proportional to domain boundary length, not domain area
- **`leaf_graph` returns per-edge spacing**, not per-leaf: matches Dijkstra weight expectation in `octree_medial.medial_distance`

## Complexity improvements delivered

| Operation | Before | After |
|---|---|---|

| 2:1 balancing | O(N³) rebuild per split | O(N log N) work-queue FIFO |
| Point location | O(N) linear scan | O(log N) tree descent |
